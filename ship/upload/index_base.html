Collaborative Ecological Modelling

Universal Simulator

  • Users: Download Windows installation file for UniSim ver. 1.1
    1. Run installation file
    2. See user’s readme file to get started
  • Developers: Download latest development snapshot for UniSim (version version_base)
    1. Unpack to your development folder. You will get a folder with C++ source code ready for the Qt Creator environment.
    2. See developer’s readme file for configuration
    3. Git user? Visit UniSim at GitHub
  • Wanted: Developers for maintaining Linux and Mac OS installation files for UniSim

Contact

  • Project Manager: Niels Holst, Aarhus University, Denmark

History

Universal Simulator grew out of efforts to develop a generic framework for modelling weed population dyncamics (WeedML). The contents of that web site (www.weedml.org) is under migration to this site.
